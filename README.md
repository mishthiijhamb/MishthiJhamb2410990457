## Steps to Run the Project  
1. **Clone the repository**  
   ```sh  
   git clone https://github.com/your-username/todo-list.git  
   cd todo-list  
   ```  

2. **Install dependencies (if any)**  
   ```sh  
   pip install -r requirements.txt  
   ```  

3. **Run the project**  
   ```sh  
   python main.py  
   ```
